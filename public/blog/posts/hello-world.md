# 欢迎来到 SAKURAIN 博客

这是我的第一篇博客文章，很高兴你来到这里！

## 关于这个博客

这个博客系统基于以下技术栈构建：

- **React 19** - 使用最新的 React 特性
- **TypeScript** - 类型安全的开发体验
- **Vite** - 极速的构建工具
- **Framer Motion** - 流畅的动画效果
- **Tailwind CSS** - 实用优先的 CSS 框架

## 设计理念

### 1. 极简主义

保持界面简洁，专注于内容本身，让读者能够轻松阅读和理解。

### 2. 性能优先

使用懒加载、代码分割和优化的构建配置，确保快速加载。

### 3. 响应式设计

完美适配桌面、平板和移动设备。
